# Portfolio Backend Server

This sample backend server is provided to demonstrate API, contact form, and status endpoints for the portfolio.

## Install

1. Open a terminal in `c:\Users\talkt\OneDrive\Desktop\Portfolio.2\backend`
2. Run `npm install`

## Start the server

Run:

```bash
npm start
```

Then open `http://localhost:3000` to view the portfolio and use the sample backend endpoints:

- `GET /api/status`
- `GET /api/projects`
- `POST /api/contact`
